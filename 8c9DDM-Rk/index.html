2023-02-13

在github看到个微信信息抓取的红队工具，挺有意思，看了看源码，打算写个给它写个 yara 规则。

项目: Sndav/WeChatDB-Rust

分析

  • 找找明文字符串
    27行的 SEARCH_STR 参数，是用作定位内存中的公私钥，可以用作特征之一。

155 行的 WeChatWin.dll 用作匹配 上文获取的指定模块的名称。

  • 找编译信息
    Cargo 是 Rust 的构建系统和包管理器
    一般来说 rust 编译的程序，里面都带有 .cargo 相关的编译信息。

  • 最后再加上文件pe文件头
    {4D 5A}
